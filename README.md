# description
игра на запонимание пар карточек. Есть три уровня сложности:
- 1 уровень - 3 пары
- 2 уровень - 6 пар
- 3 уровень - 9 пар

# rules
Нужно отгадать все пары карт чтобы выиграть. Если пара не угадана. то игра заканчивается и игрок проигрывает.
